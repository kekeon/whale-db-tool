## Features

1. 数据库切换功能
2. 栏位信息（字段信息）
3. 索引信息（找出多余索引）
4. DDL 信息
   ---表操作
5. 选中数据导出（excel\json), 选中导出的字段
6. 复制数据-复制单元格到剪切板\所选行\所有行
7. 插入新行数据（一次只能插入几行）\删除行
8. 导入数据已经导出的数据
9. 将某一列转化成表单视图
10. 每次单词查询 1000 行数据，可以进行翻页，设置行数
11. 自定义筛选
12. json 显示
